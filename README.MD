# Промышленный 3D-Измеритель на базе СТЕРЕОПАРЫ (DEXP MR1001)

Проект автоматического бесконтактного измерения габаритов промышленных объектов и пиломатериалов с использованием стереозрения OpenCV и нейросети YOLOv8.

---

## 🛠 Текущие результаты за 1-й день
1. **Калибровка доведена до идеала**: Ошибка стерео-репроекции снижена с **2-3 px** до **0.5019 px** с помощью разработанного жадного "алгоритма-киллера" кадров.
2. **Программная синхронизация**: Реализовано автоматическое выравнивание яркости и контраста между неидентичными сенсорами камер на лету.
3. **3D Визуализация**: Написан стример плотного облака точек (Point Cloud) в Open3D.
4. **Интеграция с ИИ**: Подключена нейросеть YOLOv8-segmentation для автоматического выделения контуров объектов и пересчета пиксельных габаритов в честные миллиметры по опорной плоскости.

---

## 📂 Структура проекта
* `01_collect_calibration.py` — Скрипт сбора стереопар для калибровки с UI-подсветкой углов и защитой от перезаписи.
* `02_calibrate_stereo.py` — Финальный калибратор со стабильными флагами (без взрывоопасного Rational Model).
* `03_analyze_dataset.py` — Leave-One-Out анализатор кадров.
* `04_auto_cleaner.py` — "Жадный" автоочиститель датасета от смазанных и дефектных кадров.
* `05_depth_map.py` — Тюнер полуглобального матчера (StereoSGBM) с ползунками.
* `06_stereo_measurer.py` — Интерактивный измеритель глубины по клику мыши с медианным оконным фильтром 7x7.
* `07_auto_size_validator.py` — Автоматический замер ребер калибровочных квадратов шахматной доски для выявления слепых зон.
* `08_3d_point_cloud.py` — Генератор статичного цветного 3D-облака точек в Open3D.
* `09_live_3d_cloud.py` — Быстрый стример живого 3D-потока в реальном времени.
* `10_yolo_3d_measurer.py` (или `yolo_test.py`) — Финальный промышленный измеритель габаритов объектов по опорной плоскости с использованием YOLOv8.

## 🚀 Как запустить проект

1. **Подготовка окружения:**
   ```bash
   pip install opencv-python opencv-contrib-python ultralytics open3d numpy
   ```

2. **Фиксация параметров камер в Linux (Удаление светомузыки):**
   ```bash
   v4l2-ctl -d /dev/video0 -c auto_exposure=1 -c white_balance_automatic=0
   v4l2-ctl -d /dev/video2 -c auto_exposure=1 -c white_balance_automatic=0
   ```

3. **Запуск промышленного измерителя:**
   * Замерьте рулеткой расстояние от камер до стола/стены и впишите его в `yolo_test.py` в переменную `DISTANCE_TO_BACKGROUND_CM`.
   * Запустите скрипт:
     ```bash
     python yolo_test.py
     ```
   * Направьте камеру, дождитесь обводки объекта и нажмите **ПРОБЕЛ** для вывода размеров.
